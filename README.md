# Projet Deep Learning - Analyse de plaies

Squelette du projet.

## Structure
- `data/`        : données (raw, processed, base médicale)
- `core/`        : logique métier (modèles, autoencoder, Grad-CAM, RAG, LLM...)
- `app-streamlit/` : interface Streamlit
- `scripts/`     : scripts utilitaires (drift monitoring)
- `tests/`       : tests unitaires
- `models/`, `mlruns/`, `reports/` : artefacts (non committés)
